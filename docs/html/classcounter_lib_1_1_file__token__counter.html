Detailed Description

◆ add_to_map()

void counterLib::File_token_counter::add_to_map ( Token_count_map &  tcm)

Tokenizes the file of this object. Adds the unique tokens and their frequency counts to the given Token_count_map. Always starts at beginning of the open file of this object, even when called again.

Tokenizing here is defined as separating the file into words (characters delimited by whitespace), stripping leading and trailing punctuation, and forcing the token to be lowercase.

Parameters
tcm[IN] A Token_count_map for accumulating token counts.
